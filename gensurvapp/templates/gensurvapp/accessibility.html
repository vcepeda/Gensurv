Accessibility Statement

In compliance with Section 10 Paragraph 1 of the State Disability Equality Act, the University Hospital and Medical Faculty of the Eberhard Karls University of TÃ¼bingen works to make its website accessible without obstacles (L-BGG).

This accessibility statement applies to the main UKT website at the following link: https://www.medizin.uni-tuebingen.de/de/

Status of Compatibility with the Requirements

This website is only partially compatible with Section 10(1) of the L-BGG due to the following incompatibilities:

Non-Accessible Content

The content listed below is not accessible for the following reasons:

a) Incompatibility with Section 10(1) L-BGG

  • BITV 9.1.1.1 a: Several control elements lack alternate texts
  • BITV 9.1.1.1 b: A few incorrect and many missing alternative texts for graphics
  • BITV 9.1.1.1 c: Layout components are not used with empty alt attributes
  • BITV 9.1.2.3: Missing audio description or full text alternative for videos
  • BITV 9.1.2.5: Missing audio description for videos
  • BITV 9.1.3.1 a: Incorrectly used HTML structural elements for headings
  • BITV 9.1.3.1 b: No correct HTML structure elements for lists
  • BITV 9.1.3.1 d: Structural elements are not correctly used to structure the content
  • BITV 9.1.3.1 d: For layout tables, no structural markup is used
  • BITV 9.1.3.1 e: Form element labeling cannot be determined programmatically
  • BITV 9.1.3.5: User data entry fields do not indicate their intended use
  • BITV 9.1.4.1: Not entirely usable without colors
  • BITV 9.1.4.3: Contrasts of texts not sufficient in the ratio 1:4.5
  • BITV 9.1.4.5: Partial use of font graphics
  • BITV 9.1.4.11: Contrasts for graphics and graphic control elements insufficient (1:3)
  • BITV 9.1.4.13: Faded-in contents partly not operable
  • BITV 9.2.2.2: Moving content cannot be switched off
  • BITV 9.2.4.7: Current position of focus partly not clear
  • BITV 9.2.5.3: Visible caption not always part of accessible name
  • BITV 9.3.1.1: Main language not indicated on some pages
  • BITV 9.3.1.2: Other language not marked
  • BITV 9.3.2.3: Navigation inconsistent on a few pages
  • BITV 9.3.3.1: Insufficient error detection
  • BITV 12.1.1: Missing documentation of compatibility and accessibility

Additionally, all PDFs included on the website are not accessible according to the PAC standard.

There is a lack of description in plain language and, if applicable, a variant in sign language.

b) Disproportionate Burden

The barrier-free implementation of the PDFs contained on the website and the creation of the discussion in plain language currently represent a disproportionate burden. They are therefore an exception.

c) Content Outside Applicable Legislation

A version in sign language is not mandatory.
